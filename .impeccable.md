# Design Context

## Target Audience
A single Chinese-speaking user learning English speaking through shadow reading (影子跟读法). Uses this tool daily at their desk for focused 15-30 minute practice sessions. Technically proficient but the tool should feel effortless.

## Use Cases
- Daily guided practice sessions following a 5-phase method (blind listen, read, shadow, record, retell)
- Importing English learning materials (YouTube, local files)
- Tracking practice streaks and progress over weeks/months
- Recording voice and comparing against native audio

## Brand Personality / Tone
**Studio instrument** — like a high-end audio workstation meets a meditation app. Calm, focused, professional. The interface should disappear when you're practicing and reappear when you need it. Not playful, not corporate — serene and purposeful.

## Aesthetic Direction
- Deep, warm-dark backgrounds (not cold/clinical black)
- Muted teal accent that feels like deep water — calming, not neon
- Typography-forward: English learning content displayed large and readable
- Chinese UI chrome is small, functional, secondary
- Spatial rhythm: generous whitespace around practice content, tight grouping for controls
- Subtle depth through layered surfaces, not drop shadows
- Phase colors as accent markers, not loud badges
